# miniTP Path

1. Afficher la variable d'environnement `PATH`.
2. Ajouter un chemin vers le dossier Téléchargements au `PATH` (utilisez "`:`" pour séparer la liste des répertoires).
3. Revérifier le `PATH`
4. Fermer le terminal
5. Rouvrir le terminal. Qu'est-il advenu du `PATH` ?
6. Ouvrir le fichier `~/.bashrc` et ajouter la commande du 2. à la fin du fichier
7. Rouvrir le terminal. Qu'est-il advenu du `PATH` ?
