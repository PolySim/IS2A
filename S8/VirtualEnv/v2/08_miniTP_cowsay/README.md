# miniTP cowsay

1. Créez un environnement virtuel avec `venv`
2. Activez le
3. Installez le package python `cowsay`
4. Testez `cowsay` en lui faisant dire "Bonjour !"
5. Changez le personnage en regardant la liste des "Available Characters" avec l'option `-h` et en l'activant avec `-c character`

```bash
(venv) simondesdevises@95 08_miniTP_cowsay % cowsay -t hello | lolcat
```
