Test du χ2 d’indépendance

Le test du χ2 vérifie si deux variables qualitatives sont indépendantes.

  • Hypothèses: H0 (indépendance), H1 (dépendance).
  • Statistique: somme des (observé − attendu)² / attendu ~ χ2 avec (r−1)(c−1) ddl.
  • Décision via la p-value.

p-value = 0 ; On rejette H0 : dépendance entre Region et Liste.

Solutions aux problèmes identifiés

Si vous obtenez une p-value de 0 et des valeurs propres très petites, voici les solutions :

1. Filtrage des données

# Filtrer les listes avec moins de 1000 voix
cont_table_filtered <- cont_table[, colSums(cont_table) >= 1000]
print("Dimensions après filtrage:")
print(dim(cont_table_filtered))

# Refaire le test chi2
chi2_filtered <- chisq.test(cont_table_filtered)
print(paste("P-value après filtrage:", chi2_filtered$p.value))

# Refaire l'AFC
afc_filtered <- CA(cont_table_filtered, graph = FALSE)
print("Valeurs propres après filtrage:")
print(afc_filtered$eig[1:5,])

2. Test exact de Fisher (si effectifs trop faibles)

# Pour les petits tableaux
fisher_test <- fisher.test(cont_table)
print(paste("P-value Fisher:", fisher_test$p.value))

3. Vérification de la structure des données

# Vérifier si les données sont bien structurées
print("Nombre de régions:", nrow(cont_table))
print("Nombre de listes:", ncol(cont_table))
print("Ratio:", ncol(cont_table)/nrow(cont_table))
